Simditor 初始化之后，编辑器实例会暴露一些公共方法供调用：

# 初始化 Simditor
editor = new Simditor
  textarea: $('#editor')

# 调用 setValue 方法设置内容
editor.setValue 'hello world'

公共方法

setValue val String

设置编辑器的正文内容，Simditor 会自动格式化新设置的内容。

getValue 返回值: String

用于获取编辑器的正文内容，返回值为编辑器正文的 HTML 内容。

sync 返回值: String

将编辑器的正文内容同步到 textarea 元素的 val 属性，返回值为编辑器正文的 HTML 内容。

focus 返回值: jQuery Object

用于把焦点移动到编辑器，触发 focus 事件。

blur 返回值: jQuery Object

用于让编辑器失去焦点，触发 blur 事件。

hidePopover 返回值: jQuery Object

用于隐藏编辑器界面中的 popover 弹出窗，如编辑图片、链接的弹窗。

destroy

恢复编辑器到初始化之前的状态。
