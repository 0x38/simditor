默认配置

Simditor 构造函数创建一个新的编辑器实例。通过传递的 options 参数定制实例。下面的示例使用所有选项及其默认值：

editor = new Simditor({
  textarea: null
  placeholder: ''
  defaultImage: 'images/image.png'
  params: {}
  upload: false
  tabIndent: true
  toolbar: true
  toolbarFloat: true
  pasteImage: false
})

配置选项

textarea jQuery Object / HTML Element / Selector String 默认值: null

指定用来初始化编辑器的 textarea 元素，可以接受 jQuery Object、HTML Element 或者 Selector String 类型的参数。

placeholder String 默认值: ''

编辑器的 placeholder，如果为空 Simditor 会取 textarea 的 placeholder 属性。

toolbar Boolean / Array 默认值: true

是否显示工具栏按钮，可通过数组的形式配置具体显示按钮，支持以下按钮：

[
  'title'          # 标题文字
  'bold'           # 加粗文字
  'italic'         # 斜体文字
  'underline'      # 下划线文字
  'strikethrough'  # 删除线文字
  'ol'             # 有序列表
  'ul'             # 无序列表
  'blockquote'     # 引用
  'code'           # 代码
  'table'          # 表格
  'link'           # 插入链接
  'image'          # 插入图片
  'hr'             # 分割线
  'indent'         # 向右缩进
  'outdent'        # 向左缩进
]
toolbarFloat Boolean 默认值: true

是否让工具栏按钮在页面滚动的过程中始终可见。

defaultImage String 默认值: images/image.png

编辑器插入混排图片时使用的默认图片。

tabIndent Boolean 默认值: true

是否在编辑器中使用 tab 键来缩进。

params Object 默认值: { }

键值对，在编辑器中增加 hidden 字段（input:hidden），通常用于生成 form 表单的默认参数。可生成如下标签：

<input type="hidden" name="key" value="val" />
upload Boolean / Object 默认值: false

false 或者键值对，编辑器上传本地文件的配置，键值对可以设置的参数有：

  • url 文件上传的接口地址；
  • params 键值对，指定文件上传接口的额外参数，上传的时候随文件一起提交；
  • connectionCount 同时上传文件的最大数量；
  • leaveConfirm 在文件上传过程中离开页面时提醒的文案。

键值对参数的默认值：

editor = new Simditor({
  textarea: $('#editor')
  upload:
    url: ''
    params: null
    connectionCount: 3
    leaveConfirm: '正在上传文件，如果离开上传会自动取消'
})
pasteImage false 默认值: Boolean

是否允许粘贴上传图片，依赖 upload 选项，仅支持 Firefox 和 Chrome 浏览器。
